## Éléments manquants à demander au client

### Histoire
- [ ] Quand le laboratoire a-t-il été fondé?
- [ ] Par qui? (Chérine seule ou avec partenaires?)
- [ ] Pourquoi avoir créé Quintessence? Quelle était la motivation?
- [ ] D'où vient le nom "Quintessence"?

### Mission & Vision
- [ ] En une phrase, quelle est votre mission?
- [ ] Où voyez-vous Quintessence dans 5-10 ans?
- [ ] Qu'est-ce qui vous motive chaque jour?

### Valeurs
- [ ] Quelles sont vos 3-5 valeurs principales?
- [ ] Exemples: Excellence, Innovation, Art, Précision, Service personnalisé?

### Différenciateurs
- [ ] Qu'est-ce qui vous distingue des autres labos dentaires?
- [ ] Pourquoi un dentiste devrait choisir Quintessence?
- [ ] Avez-vous des certifications, accréditations, prix?
- [ ] Technologies exclusives ou rares que vous utilisez?

### Chiffres / Crédibilité
- [ ] Années d'expérience combinée de l'équipe?
- [ ] Nombre de cas traités / prothèses réalisées?
- [ ] Nombre de clients (cliniques/dentistes)?

### Témoignages
- [ ] Avez-vous des témoignages de dentistes satisfaits?
