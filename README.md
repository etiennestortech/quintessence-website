# Website Onboarding

Structure de projet pour onboarding clients web.

## Structure

```
/templates/              # Templates réutilisables
  QUESTIONNAIRE-DISCOVERY.md
  CLIENT-INFO-TEMPLATE.md
  ACTION-ITEMS-TEMPLATE.md

/projects/               # Projets clients
  /quintessence/         # Laboratoire dentaire
  /depanneur-beausejour/ # Dépanneur
```

## Nouveau projet

1. Créer un dossier dans `/projects/`
2. Copier les templates nécessaires
3. Remplir le questionnaire pendant le discovery call
4. Compléter CLIENT-INFO après l'appel

## Templates disponibles

- **QUESTIONNAIRE-DISCOVERY.md** - Questions pour discovery call
- **CLIENT-INFO-TEMPLATE.md** - Infos client et projet
- **ACTION-ITEMS-TEMPLATE.md** - Suivi des tâches
